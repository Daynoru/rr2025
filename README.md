### Основные команды
- pwd Где я? 
- cd Перейти в другое расположение 
*. = начало относительного пути*
* .. = подняться на уровень выше*
* ~ = домашняя папка*
* / = разделитель уровней*
* Tab = автозаполнение*
- mkdir Создать пустую папку 
- ls Посмотреть содержимое папки
- ls -lh /home/user/test
* -h удобное отображение размера*
* -t сортировка по размеру*
* -l табличкой*
- head, tail Посмотреть начало или конец файла
- cat Посмотреть весь файл
- grep Поиск текста в файле
- | Передать вывод одной команды другой команде (pipe)
- >  Записать вывод команды в файл 
* > file.txt*
- mv, cp Переместить или переименовать; копировать 
* -r = работа с папкой*
- rm Удалить 
*-r = удаляет папку*
- nano текстовый редактор
